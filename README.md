## 📋 Sobre o Projeto

Landing page profissional e moderna desenvolvida para o **Grupo LF**, empresa de segurança privada com 15 anos de atuação em Pernambuco. O site foi criado seguindo as melhores práticas de design moderno, com foco em conversão e experiência do usuário.

### 🎯 Objetivo

Apresentar os serviços de segurança privada do Grupo LF de forma profissional, atrativa e persuasiva, facilitando o contato direto com clientes potenciais através de múltiplos canais de comunicação.

---

## ✨ Características Principais

### 🎨 Design Moderno
- **Paleta de cores profissional**: Azul escuro (#0d1b2a), azul médio (#1e4a8a), verde neon (#00e676) e dourado (#ffd700)
- **Tipografia elegante**: Google Fonts Inter com pesos variados
- **Layout limpo e organizado**: Espaçamentos generosos e hierarquia visual clara
- **Elementos visuais SVG**: Ícones e ilustrações vetoriais inline para performance

### 📱 Totalmente Responsivo
- **Mobile-first**: Design otimizado para dispositivos móveis
- **Breakpoints estratégicos**: 480px, 768px, 1024px
- **Menu hamburger**: Navegação intuitiva em telas pequenas
- **Imagens adaptativas**: SVGs escaláveis sem perda de qualidade

### ⚡ Performance Otimizada
- **Sem dependências externas**: HTML, CSS e JavaScript puro
- **Animações suaves**: Intersection Observer para fade-in na rolagem
- **Lazy loading**: Preparado para carregamento progressivo de conteúdo
- **Debounce em eventos**: Otimização de scroll e resize

### 🚀 Funcionalidades Interativas
- **Scroll suave**: Navegação interna com animação
- **Animações ao scroll**: Elementos aparecem conforme o usuário rola a página
- **Menu mobile funcional**: Hamburger menu com animação
- **Header dinâmico**: Sombra aumenta ao rolar a página
- **WhatsApp flutuante**: Botão fixo de contato em todas as páginas

## 🚀 Como Usar

### Opção 1: Abrir Localmente
1. Faça o download dos arquivos do projeto
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. Pronto! O site estará funcionando localmente

### Opção 2: Publicar Online
1. Faça upload dos arquivos para um servidor web ou serviço de hospedagem
2. Mantenha a estrutura de pastas intacta
3. Acesse através do domínio configurado

### Opção 3: Usar a aba Publish
1. Clique na aba **Publish** desta ferramenta
2. Publique o projeto com um clique
3. Receba a URL pública imediatamente
